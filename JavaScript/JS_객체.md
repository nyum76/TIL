# 인터넷프로그래밍 7주차 : JavaScript
## 객체
* 하나의 배열에 복합 데이터 저장
* 키-값 쌍으로 복합 정보 저장
* 객체 저장
  * `{"키1":"값1","키2":"값2",...}`
    * 예 
        ```js
        var student = {
            'name':'홍길동','id','98e08032','gender':'male'
        }
        ```
* 객체 요소 접근
  * `객체명.키`
    * 예
    ```js
    student.name // 학생의 이름 가져오기
    ```
  * 괄호 표기법으로 객체의 프로퍼티 접근
    * `객체명['프로퍼티명']` 
* 프로퍼티 : 객체의 한 키-값 쌍
