# 기본 선택자 사용하기

## 전체 선택자
```css
*{/* CSS 코드 */}
```
- 모든 요소 한 번에 선택자로 지정
---
## 태그 선택자
```css
태그명{/* CSS 코드 */}
```

---

## id 선택자
```css
#id속성값{/* CSS 코드 */}
```

---
## 클래스 선택자
```css
.class속성값{/* CSS 코드 */}
```

---
## 기본 속성 선택자
```css
[속성]{/* CSS 코드 */}
[속성=값]{/* CSS 코드 */}
```

---
속성 선택자는 다른 선택자와 함께 사용할 수 있다
-> 같이 쓰면 둘 다 포함해야 선택

-> 같이 안쓸 경우 해당 속성만 있으면 선택
```css
다른선택자[속성선택자]{
  /* CSS 코드*/
}
```
