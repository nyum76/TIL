# CSS

## CSS 단위
- `px` : 픽셀
  - 절대 단위
- `%` : 상대적 백분율
  - 특정 값을 기준으로 몇 퍼센트의 크기를 가지는지
- `em` : 요소의 글꼴 크기
  - 요소의 글꼴 크기가 `16px` 인 경우, `1rem=16px`
- `rem` : 루트(최상위) 요소의 글꼴 크기
- `vw` : 뷰포트 가로 너비의 백분율
  - 뷰포트의 가로 너비가 `1920px` 인 경우, `10vw= 192px`
- `vh` : 뷰포트 세로 너비의 백분율
  - 뷰포트의 세로 너비가 `1080px` 인 경우, `10vw= 108px`
